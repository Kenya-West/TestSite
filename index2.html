Office Online

Сохраняйте документы, таблицы и презентации в веб-хранилище OneDrive. Делитесь ими и работайте над ними одновременно с другими людьми. Начните сейчас, это бесплатно!

Office Online объединяет наиболее часто используемые приложения Office и предоставляет возможность совместного редактирования в реальном времени. При помощи Office Online вы сможете бесплатно работать вместе с друзьями или членами семьи над общими документами, презентациями, электронными таблицами и записными книжками.

  • Создавайте документы, электронные таблицы и презентации в Интернете или с помощью классической версии Office
  • Сохраняйте их в онлайн-хранилище OneDrive
  • Делитесь с другими пользователями и работайте вместе в реальном времени

Перейдите на страницу Office.com и начните работу с Office Online, воспользовавшись своей учетной записью Microsoft.

Windows Phone 8

20 июня 2012 года на организованной Microsoft конференции под названием Windows Phone Summit была анонсирована Windows Phone 8. Главное преимущество восьмой версии — возможность объединить планшеты, смартфоны и персональные компьютеры в единую «экосистему», то есть возможность создания условий для разработчиков, облегчающих портирование программного обеспечения между этими устройствами. Новое поколение платформы будет основано на ядре NT и максимальное разрешение экрана аппаратов будет 1920x1080, в отличие от Windows Phone 7.x, которая основана на ядре CE с разрешением экрана 800×480.
29 октября 2012 года Microsoft официально представила следующее поколение платформы. Новшества Windows Phone 8 — Internet Explorer Mobile 10-й версии с функцией обнаружения вредоносных сайтов, поддержка многоядерных процессоров, технология обмена данными NFC, слот для карт памяти и другие. Пользователи получили возможность изменять размер «живых плиток» стартового экрана — теперь их три вида (маленькие, средние и большие).

Windows Phone 8.1

Windows Phone 8.1 — обновление мобильных устройств на базе ОС Windows Phone. Презентация нового обновления прошла 2 апреля 2014 года на ежегодном мероприятии компании Microsoft – //Build/ 2014. 14 апреля 2014 года началась рассылка Developer Preview версии. Автор сайта уже обновился, хе-хе.

Особенности Windows Phone 8.1

  • Action Center - центр уведомлений для Windows Phone 8.1.
  • Экран блокировки - теперь приложения могут не просто работать с экраном блокировки. Они могут тотально его изменять.
  • Три ряда плиток.
  • Cortana. Голосовому ассистенту было отведено много внимания. Об этом будет рассказано в отдельной статье далее.
  • Изменения в магазине. Приложение магазина было переработано, добавлены новые ссылки, новые возможности.
  • Календарь. Существенно улучшено приложение календаря. В него добавлены новые режимы просмотра, он более удобен в использовании.
  • WiFi Sense. С WiFi Sense вы получите новые возможности, которые сделают работу с WiFi более автоматизированной. При этом вам не надо буквально говорить пароль от сети.
  • Более глубока интеграция VoIP-сервисов. Во время звонка вы сможетемоментально переключиться на звонок из сотовой сети на Skype, Viber или другие.
  • Swype-клавиатура. Долгожданная функция, которая перенесет набор на клавиатуре на совершенно новый скоростной уровень.

Windows RT 8

Windows RT (ранее известная как Windows 8 ARM) — редакция операционной системы Windows 8 компании Microsoft для планшетных и других компьютеров на базе ARM-процессоров.
Windows RT не совместима с Windows-приложениями, созданными для процессоров c набором команд x86-x64 (Intel, AMD) и поставляется с планшетными компьютерами на ARM-процессорах компаний-производителей ASUS, Microsoft, Dell, Samsung и Nokia в качестве предустановленной системы.
Основным источником пополнения программ является интернет-магазин компании Microsoft Windows Store, содержащий десятки тысяч приложений, большинство из которых доступно бесплатно. Также для Windows RT доступны основные приложения и пакеты. Речь, прежде всего, о наиболее популярных Microsoft Office 2013 (Word, Excel, Outlook, OneNote, Powerpoint), Adobe Photoshop, Abbyy Finereader и ряде других.

Windows RT 8.1

Windows RT 8.1 — это операционная система семейства Windows, которая оптимизирована для тонких и легких компьютеров с увеличенным временем работы батареи. В Windows RT 8.1 работают только встроенные приложения или приложения, скачанные из Магазина Windows. Клиентский компонент Центра обновления Windows автоматически обновляет ПО на компьютере, а Защитник Windows оберегает компьютер от вирусов и вредоносных программ.
Действует бесплатное обновление до Windows RT 8.1 из Магазина Windows. Windows RT 8.1 можно установить только на компьютеры или планшеты, уже работающие под управлением Windows RT.

Outlook.com

Outlook.com — это бесплатная почтовая служба, которая предоставляет вам исключительные возможности.

Особенности Outlook.com

Общий чат для Skype, Facebook и Google
Общайтесь сразу со всеми друзьями из Skype, Facebook и Google прямо в почте.
Работа с Office
Бесплатно просматривайте и редактируйте документы Word, Excel и PowerPoint прямо в почтовом ящике. Ни вам, ни вашим адресатам не нужно устанавливать Office, и при этом форматирование документов остается без изменений.
На связи с друзьями
Просматривайте новые статусы, фотографии профиля и твиты друзей во время переписки.
Усовершенствуйте свое устройство с Windows
Outlook.com — это эффективная персонализированная почтовая служба, которая идеально подходит для Windows. Войдите в приложение "Почта" в Windows 8.1 со своей учетной записью и откройте для себя уникальные и мощные инструменты упорядочения писем, автоматическую синхронизацию с приложениями "Календарь", OneDrive, "Люди" и другие потрясающие возможности.
Оптимально для "Почты Windows"
Добавьте учетную запись Outlook.com в приложение "Почта" в Windows 8.1 и с удовольствием пользуйтесь уникальными и эффективными функциями для упорядочения писем, например "Очисткой". Таких инструментов вы не найдете в Gmail и Yahoo! Mail. Новостные рассылки и социальные оповещения упорядочиваются автоматически. Кроме того, вы можете настроить автоматические ответы на время отпуска, но только если Outlook.com используется в приложении.

OneDrive

Одно хранилище абсолютно для всего
Легко сохраняйте фотографии, видео, документы и другие файлы, а затем делитесь ими — когда угодно, на любом устройстве и совершенно бесплатно. Кроме того, получите 7 ГБ при регистрации.
Одно хранилище для всех ваших фотографий
Ваши любимые фотографии и видео будут в безопасности, даже если с устройством что-нибудь случится. Получите дополнительно 3 ГБ, активировав автоматическое резервное копирование фотографий с камеры. Затем просматривайте свои фотографии в любое время на любом устройстве.
Одно хранилище для всех ваших документов
OneDrive работает с Office, поэтому создавать, редактировать документы и делиться ими очень легко. Сохраняйте документы в OneDrive и работайте с ними на любом устройстве.
Одно хранилище для совместного доступа к всевозможным файлам
Поделитесь одним файлом или целым фотоальбомом — без вложений. Отправляйте сообщения со ссылками на файлы, которыми нужно поделиться, сохраняя конфиденциальность другой информации, хранящейся в OneDrive.
OneDrive абсолютно для всего
Храните свои любимые фотографии, видео, документы, файлы и многое другое и делитесь ими в OneDrive. Зарегистрируйтесь и получите бесплатно 7 ГБ.
